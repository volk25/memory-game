#### Memory Game 1.0 ####This application was developed completely in Javascript.####This game is aimed to test and train the memory and attention of the player.  On the screen user will see 25 purple squares. Some of the squares will turn yellow for a few seconds. Player has to remember which square was yellow and click on it. With every correct response, the level of the game increases and more squares will turn yellow at the same time.  For every correct square player gets 1 point.  The goal is to get the highest score as possible.####This game was developed for educational and entertainment purposes.  ####©ÊCopyright Nikita Volkov 2021
